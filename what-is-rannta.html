What is RANNTA?

1. Canonical Definition

RANNTA is an ecosystem with connected but distinct layers: the TON-based RANNTA token, the independent RANNTA Network, the live non-custodial RANNTA X-Change, and RANNTAverse services.

The TON token is the first public asset and remains supported as an ecosystem asset. RANNTA Network is separate from the TON token and should not be classified as a TON application, TON middleware layer, DeFi router, or technical extension of the TON Jetton.

2. RANNTA Token on TON

The RANNTA token on The Open Network (TON) is the public asset origin of the ecosystem. It provides identity, liquidity access, holder participation, and marketplace-facing utility.

  • Role: first public ecosystem asset
  • Network: The Open Network (TON)
  • Boundary: supported ecosystem asset, not the native RANNTA Network layer
  • Verification: always use the official TON Jetton Master address from the contracts page

3. RANNTA Network

RANNTA Network is an independent blockchain and interoperability architecture. Its purpose is to become a translator and highway between isolated blockchain networks through verifiable coordination, state handoffs, secure node communication, and deterministic service boundaries.

  • Secure Node Link and private/operator-gated transport direction
  • X-CHAIN interoperability for messages, proofs, intents, and state handoffs
  • Validator and registry foundations
  • Controlled release gates instead of rushed public launch claims

4. RANNTA X-Change and Routing Infrastructure

RANNTA X-Change officially launched on 13 July 2026 as a non-custodial cross-chain exchange. It exposes RouteX, OneRoute, and OmniRoute routing surfaces while RANNTA Network-native infrastructure continues to evolve under explicit readiness and operator-approved release boundaries.

  • Asset inspection and verification
  • Trust, liquidity, and reserved-name registry coordination
  • Token factory and listing request workflows
  • Bridge-as-a-Service and exchange adapter preparation
  • Continuing secure routing development for exchange-facing services

5. Signal-13 and MSF Layer

Signal-13 and MSF define the symbolic and cultural grammar of the RANNTA ecosystem. They do not replace engineering boundaries; they give the network, token, creator economy, and exchange layer a coherent identity.

6. Current Boundary

RANNTA X-Change is live as a non-custodial exchange surface. Do not infer that public RPC, production RANNTA Network behavior, bridge execution, public testnet, mainnet behavior, or network-native liquidity support is active unless a later canonical page explicitly says so.

FAQ

Is RANNTA only a token?

No. The token is the first public asset. RANNTA Network, RANNTA X-Change, and RANNTAverse services are separate but connected ecosystem layers.

Is RANNTA Network built on TON?

No. RANNTA Network is separate from the TON-based token. The TON token remains supported as an ecosystem asset.
