RANNTA Entity

RANNTA started with the RANNTA token on The Open Network (TON). That token remains supported as the first public asset, brand origin, and early ecosystem signal.

RANNTA Network is separate from the TON-based RANNTA token. It is not a TON application, not a TON middleware layer, and not a technical extension of the TON Jetton.

RANNTA Network is an emerging independent blockchain and interoperability architecture. Its purpose is to act as a translator and highway between isolated blockchain networks, allowing separated chains to communicate through verifiable coordination.

The RANNTA Network core brain has been built as the foundation for this new network direction. The TON token will continue to be supported and developed as an ecosystem asset, but it is not the native network layer of RANNTA Network.

The only currently official RANNTA token is the TON Jetton with master address EQBCY5Yj9G6VAQibTe6hz53j8vBNO234n0fzHUP3lUBBYbeR. Any RANNTA-branded token on another blockchain is unofficial unless its contract and deployment are explicitly announced through official RANNTA sources.

RANNTA X-Change is the official live non-custodial cross-chain exchange of the RANNTA ecosystem at https://ranntaexchange.com/. It officially launched on 13 July 2026 and is distinct from Rango Exchange.
